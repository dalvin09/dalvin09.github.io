Concept#

The Profile Switch is the highest-conviction setup in the playbook. It is built on the idea that when a profile closes extended — with single prints stretching into, through, or beyond a reference area (or to a relative extreme) — and the next profile switches (at 3am or 11:30am) and confirms that extension by holding the previous profile’s close, halfback, or HVN when tested, the market is telling you something unambiguous: the move is real and has been accepted.

Value building or one-time framing in the new profile’s direction seals it.

This setup is as nuanced as it is profitable. Respect both sides of that equally.

Criteria (All must be met)#

  • Dollar is trending — directional context is established, not range bound
  • The current/prior profile closes extended: single prints into, through, or past a reference area — OR to a relative extreme (session high/low, weekly extreme, etc.)
  • Profile switch occurs at 3am or 11:30am
  • The new profile tests one of the following levels from the prior profile when it opens:
    • Prior profile close
    • Halfback of the prior profile
    • HVN from the prior profile
  • That test holds — the level is defended, not violated
  • New profile shows value building higher/lower in the direction of the extension, OR one-time framing building in that direction

Entry#

  • Trigger: New profile tests and holds prior close / halfback / HVN — with value or OTF confirmation building
  • Entry: At or near the tested and held level
  • Stop: Below/above the tested level — if 5 pips isn’t enough, rethink the trade
  • Target: Open-ended — trail aggressively; this is a continuation trade in a trending environment
  • Exit: Scale out as the move develops, but leave a runner — the move can go much further than expected

Rules#

  1. Dollar must be trending. No trend, no trade.
  2. The prior profile must be extended with single prints into/past a reference or to a relative extreme. A balanced close disqualifies the setup.
  3. Profile switch must be at 3am or 11:30am. These are the only valid transition windows.
  4. Wait for the new profile to test the level. Do not anticipate — let price come to you.
  5. The test must hold. A clean rejection or acceptance at the level is required.
  6. Value building or OTF in direction is required for full confirmation. One without the other is borderline — use judgment.
  7. Do not rush to exit. This setup produces moves that look ugly mid-trade. Drawdown is part of the process. The thesis is invalidated only if the level breaks, not if price wiggles.

Notes#

The nuance here is real and non-trivial. Single prints tell you the market moved without looking back. The reference area or extreme tells you where it moved to. The profile switch and level hold tell you the market accepted that move. Value building or OTF confirms who is driving it.

When all of that lines up, the trade can run far. The temptation will be to take profit early — especially when it gets uncomfortable. Resist it. The whole edge of this setup lives in holding through the noise. If the level holds and the context is intact, stay in.

The uglier it gets, sometimes the better the eventual outcome. Trust the structure.
